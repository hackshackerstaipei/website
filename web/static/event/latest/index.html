第 16 場小聚

Google 地圖工具工作坊

10月6日（日）Hacks/Hackers Taipei第16次小聚，將邀請專家手把手教你操作Google地圖工具（2D到3D衛星圖），同時還有社群之友分享TechCamp的學習心得。

Hacks/Hackers Taipei的第十六次小聚「Google地圖工具工作坊」，特別邀請社群之友關鍵評論網資料圖表組長林奕甫（If），以工作坊的形式教你活用Google地圖工具。另外，我們也邀請到曾參與AIT TechCamp的天下雜誌未來城市頻道總監陳芳毓（Siren），來分享TechCamp的學習心得。

活動流程

  • 13:20 ~ 13:30 / 報到
  • 13:30 ~ 13:40 / 開場
  • 13:40 ~ 15:40 / Google 地圖工具工作坊
  • 15:40 ~ 16:00 / 中場休息與交流
  • 16:00 ~ 16:30 / TechCamp 心得分享
  • 16:30 ~ 16:40 / Q&A 與討論

因為場地的時間有限，本次小聚取消短講一次。當天活動依舊鼓勵大家共同筆記，共筆連結在此，歡迎一起參與記錄。

報名規範

因為場地座位有限，我們不開放候補，想要參加的朋友，麻煩一定要事先報名。

另外，感謝所有人過去對我們的支持，但最近幾次小聚，慢慢出現報名者最後一刻臨時取消、或是神隱不出席。我們理解大家工作與生活繁忙，但小聚的座位都非常有限，所以Hacks/Hackers Taipei的新作法是：

希望您確定可以參與後，再報名索取票券，別讓沒搶到票的朋友難過。如果你已經報名，但卻在（1) 活動前48小時內臨時取消，或 (2) 當天未出席，將取消你報名下一次小聚的機會。謝謝你的支持，期待你活動當天的加入！
